ViDER is a free, web-based data visualization tool that interfaces with REDCap (Research Electronic Data CAPture) in a dynamic fashion. The purpose of ViDER is to provide quick data visualizations (histograms, scatterplots, parallel sets) for exploration of variables captured in REDCap. Since ViDER is web-based, no special software is needed to be downloaded or installed.

ViDER accesses the current project's data and metadata to provide the visualizations. All visualizations are done client side, and requires very minimal server side processing.
